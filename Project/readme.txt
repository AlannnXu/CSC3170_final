Query 里面是可能能用上的功能
	现在分成三类：
		1.customer：     a.获取积分超过定值的客户
				b.获取指定ID的用户具体信息
				c.获取在指定日期后办理会员的客户
				d.获取所有客户的手机号码
		2.employee：	a.员工的平均工资
				b.部门的员工总数
				c.获取超过定值工资的员工
				d.将员工按照工资升序排列
		3.good：		a.获取特定品牌的所有商品
				b.获取特定种类的所有商品
				c.库存低于一定值的所有商品
				d.在一个范围区间内的所有商品
				e.前十名销量最好的商品
				f.库存内的所有商品的总值
